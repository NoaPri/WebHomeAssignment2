# Web Home Assignment 2

This repository contains a web layout assignment built using HTML and CSS.
The project is hosted using GitHub Pages and is fully accessible on both desktop and mobile devices.

## Live Website
https://noapri.github.io/WebHomeAssignment2/

## Project Structure
The website includes a main index page with links to four different layout implementations:

1. Manual Plain CSS – written manually without Copilot  
2. Plain CSS – created using GitHub Copilot  
3. Flexbox Layout – created using GitHub Copilot  
4. Bootstrap Layout – created using GitHub Copilot  

Each page demonstrates a different approach to building layouts using web technologies.

## Technologies Used
- HTML5  
- CSS3  
- Flexbox  
- Bootstrap 5  
- GitHub Pages
